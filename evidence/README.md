# Evidencia de Actividad - Gestión de Cursos

## Funcionalidades Implementadas:

- **Backend:** Registro de rutas mediante Routers y ViewSets en Django REST Framework.
- **Frontend:** - Formulario de creación con validación (Zod y React Hook Form).
  - Listado dinámico de cursos con formateo de precios (Decimal a Moneda).
  - Integración de componentes Select (Shadcn/UI) para niveles (Básico, Intermedio, Avanzado).
  - Manejo de errores de hidratación y tipos de datos.
